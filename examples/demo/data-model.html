Joins & retrieval

A user’s order historyvia API
orders⋈order_items⋈products⋈users
The API joins orders → items → products to render a user’s purchase history in a single round trip.
SELECT o.id, o.created_at, p.name, i.qty, i.price_cents
FROM orders o
JOIN order_items i ON i.order_id = o.id
JOIN products p ON p.id = i.product_id
WHERE o.user_id = $1
ORDER BY o.created_at DESC;
Best-selling productsvia API
order_items⋈products
Aggregates line items by product to rank sellers.
SELECT p.name, sum(i.qty) AS sold
FROM order_items i
JOIN products p ON p.id = i.product_id
GROUP BY p.name
ORDER BY sold DESC
LIMIT 10;
